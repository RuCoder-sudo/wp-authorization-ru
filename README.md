# Авторизация WordPress через Яндекс и Mail.ru

<div align="center">

## О плагине

**WP Authorization RU** добавляет кнопки «Войти через Яндекс» и «Войти через Mail.ru» на все ключевые страницы входа вашего WordPress-сайта. Работает со стандартными страницами WordPress и полностью интегрируется с WooCommerce.

- 🟠 **Яндекс ID** — вход через аккаунт Яндекс
- 🔵 **Mail.ru** — вход через аккаунт Mail.ru

## Возможности

### Где появляются кнопки

| Место | Яндекс | Mail.ru |
|---|:---:|:---:|
| Страница входа WordPress | ✅ | ✅ |
| Страница регистрации WordPress | ✅ | ✅ |
| Мой аккаунт WooCommerce — вход | ✅ | ✅ |
| Мой аккаунт WooCommerce — регистрация | ✅ | ✅ |
| Оформление заказа WooCommerce | ✅ | ✅ |

### Логика авторизации

| Ситуация | Что происходит |
|---|---|
| Первый вход через OAuth | Автоматически создаётся аккаунт WordPress (роль: Подписчик) |
| Email уже зарегистрирован | Плагин находит существующий аккаунт и привязывает OAuth ID |
| Повторный вход | Распознавание по OAuth ID провайдера |
| Изменилось имя у провайдера | Имя и фамилия в профиле WordPress обновляются автоматически |

### Безопасность

| Механизм | Описание |
|---|---|
| CSRF-защита | state-параметр (WordPress nonce + transient, 10 минут) |
| Callback URL | WordPress REST API — чистый URL без query-параметров |
| Санитизация | Все входные данные проходят через встроенные функции WordPress |
| Редирект | Только на страницы того же сайта |

## Требования

- WordPress 5.8 или выше
- PHP 7.4 или выше (протестировано на PHP 8.0–8.3)
- WordPress REST API должен быть доступен (не заблокирован плагинами безопасности)
- Client ID и Client Secret от Яндекс или Mail.ru

## Установка

### Способ 1: через панель WordPress

1. Скачайте ZIP-архив плагина
2. Перейдите в **Плагины → Добавить новый → Загрузить плагин**
3. Выберите файл `wp-authorization-ru.zip` и нажмите **Установить**
4. Нажмите **Активировать плагин**

### Способ 2: через FTP / SSH

1. Клонируйте репозиторий или распакуйте архив:
   ```bash
   git clone https://github.com/RuCoder-sudo/wp-authorization-ru.git
   ```
2. Загрузите папку `wp-authorization-ru` в `/wp-content/plugins/`
3. Активируйте плагин в **Плагины → Установленные плагины**

## Настройка

### Яндекс ID

1. Откройте [oauth.yandex.ru](https://oauth.yandex.ru/) → «Зарегистрировать приложение»
2. Тип платформы: **Веб-сервисы**
3. Права доступа: `login:info`, `login:email`
4. **Redirect URI** — скопируйте из **Настройки → Авторизация RU** (кнопка «Копировать»):
   ```
   https://ВАШ-САЙТ.ru/wp-json/wp-auth-ru/v1/yandex/callback
   ```
5. Скопируйте **Client ID** и **Client Secret** → вставьте в настройки плагина
6. Включите галочку «Включить Яндекс OAuth» → **Сохранить**

### Mail.ru

1. Откройте [o2.mail.ru/app](https://o2.mail.ru/app/) → «Добавить приложение»
2. Тип приложения: **Сайт**
3. **Redirect URI** — скопируйте из **Настройки → Авторизация RU**:
   ```
   https://ВАШ-САЙТ.ru/wp-json/wp-auth-ru/v1/mailru/callback
   ```
4. Скопируйте **Client ID** и **Client Secret** → вставьте в настройки плагина
5. Включите галочку «Включить Mail.ru OAuth» → **Сохранить**

> ⚠️ **Важно:** каждому провайдеру нужно своё отдельное приложение и свой Redirect URI. Не вводите два URL в одно поле.
